Safe examples (फक्त तुमच्या labमध्ये)

Hands-on examples (benign, permission-based)

1) Discover live hosts (Nmap ping scan)

Only on your own network/subnet.

# Debian/Ubuntu/Kali
sudo apt update && sudo apt install nmap

# Safe ping scan (no port probing)
sudo nmap -sn 192.168.1.0/24

2) Check services on your own device

# Replace with your host IP
sudo nmap -sV 192.168.1.10

3) View listening ports on your machine

sudo ss -tulpen

4) Capture small packet sample (own interface)

# Replace wlan0/eth0 with your interface
sudo tcpdump -i wlan0 -c 25 -vv
Ethics & legality: Test only in controlled labs or with explicit permission. उद्देश: शिकणे आणि सुरक्षा सुधारणा.
Workflow (शिकण्याची पायरी)

Suggested learning path

  1. Set up a lab: VMs/spare devices तुम्ही control करता.
  2. Linux basics: Shell, users, networking (ip, ss, journalctl).
  3. Tool practice: Nmap, tcpdump, Wireshark labमध्ये.
  4. Automation: Bash/Python scriptsने routine checks.
  5. Document & remediate: Findings नोंदवा; hardening steps घ्या.
